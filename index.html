Tentang Saya
Mengapa IT Kesehatan?

Saya adalah seorang Developer .NET Fullstack yang mendedikasikan keahlian saya untuk membangun ekosistem teknologi kesehatan (Healthcare IT) di Indonesia. Bagi saya, menulis kode bukan sekadar menyelesaikan baris perintah, melainkan memastikan bahwa sistem informasi rumah sakit (SIMRS) berjalan lancar tanpa kendala demi keselamatan pasien dan kemudahan tenaga medis.

Dengan pengalaman lebih dari 4 tahun, saya terbiasa menghadapi tantangan kritis seperti integrasi API BPJS Kesehatan, sinkronisasi RME (Rekam Medis Elektronik) SATUSEHAT Kemenkes RI, hingga mengimplementasikan kecerdasan buatan (AI) untuk diagnosis klinis awal. Saya percaya teknologi adalah kunci dalam mempercepat kemajuan pelayanan kesehatan nasional.

Prinsip Kerja Saya
  • Keandalan Tinggi: Sistem medis harus aktif 24/7 tanpa celah kesalahan.
  • Keamanan Data: Perlindungan maksimal untuk rekam medis pasien sesuai standar nasional.
  • Efisiensi Proses: Mengotomatiskan proses pendaftaran & pelayanan agar dokter fokus pada pasien.
